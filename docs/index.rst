This package has a command-line interface.


Installation
------------

To install Dress up,
run this command in your terminal:

.. code-block:: console

   $ pip install dressup


Usage
-----

Dress up's usage looks like:

.. code-block:: console

   $ dressup [OPTIONS]

.. option:: --version

   Display the version and exit.

.. option:: --help

   Display a short usage message and exit.
